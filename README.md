# Tech Website (HTML, CSS, JavaScript)

This project is a fully responsive and interactive tech website built with HTML, CSS, and JavaScript in separate files. The site includes product showcases, customer reviews, contact details, animations, and a light/dark mode feature. It is designed to work smoothly across mobile, tablet, and desktop devices.

Features
1. Modern UI

Clean and structured layout

Dedicated sections for hero, product display, testimonials, pricing, about, and contact

2. Responsive Design

Layout adjusts smoothly for phones, tablets, and desktops

Built using custom CSS and media queries

3. Light and Dark Mode

Theme toggle button

User preference saved using localStorage

4. Smooth Interactivity

Hover animations

Scroll animations

Interactive navigation bar

Contact form with basic JavaScript validation

5. Reusable & Semantic Code

Organized HTML structure

CSS separated for styling

JavaScript handles all interactions and behaviors

Technologies Used

HTML5

CSS3 (custom styling + media queries)

JavaScript (ES6) for toggles, animations, and interactivity
